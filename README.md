# Hamburgueria

## SOBRE

Este projeto se refere a um trabalho da faculdade que tem o objetivo de representar um CRUD de pedidos de uma hamburgueria, sendo uma aplicação apenas Front-End.

## TECNOLOGIAS

- CSS
- HTML
- JavaScript
